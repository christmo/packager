Configuración


Para utilizar xplook render se debe agregar el script en la cabecera del archivo html asi:


<script type="text/javascript" charset="utf-8" src="js/xplookRender.js"></script>

Del lado del servidor debe incluir el archivo

packager.jar

En el archivo js puede encontrar cambiar la configuracion del render como cambiar de template, habilitar la salida de consola para herramientas de desarrollo en navegadores, agregar un control nuevo durante la carga. etc


				xplook = {
					version: " Xplook Render - 1.0",
					template:undefined,
					splinner:true,
					restEndPoiny: 'Services/JSONEndPoint',
					controls:['top','foother','rightMenu'],
					outputMesages: true,
					};
